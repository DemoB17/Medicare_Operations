🏥 Medicare Operations Analytics Dashboard

📌 Project Overview

This project showcases an end-to-end healthcare analytics solution designed to analyze Medicare operations data and provide actionable insights through interactive dashboards. The project follows the complete data analytics workflow, from data cleaning and preparation to SQL analysis and Power BI visualization.

The dashboard enables stakeholders to monitor operational performance, patient demographics, financial metrics, physician performance, and appointment trends, supporting data-driven decision-making.

---

🎯 Business Objectives

The goal of this project is to:

- Monitor key healthcare operational KPIs.
- Analyze patient demographics and visit patterns.
- Evaluate physician performance.
- Track payment collections and outstanding balances.
- Identify trends in appointments and diagnoses.
- Support strategic and operational decision-making through data visualization.

---

🛠️ Tools & Technologies

- Microsoft Excel – Data cleaning and preparation
- SQL – Data exploration and business analysis
- Power BI – Interactive dashboard development
- DAX – Custom measures and KPIs

🔄 Project Workflow

1. Collected and reviewed the raw dataset.
2. Cleaned and standardized the data using Microsoft Excel.
3. Exported the cleaned dataset for analysis.
4. Performed data exploration and business queries using SQL.
5. Imported the cleaned dataset into Power BI.
6. Built relationships, calculated measures, and created DAX formulas.
7. Designed interactive dashboards to communicate business insights.
