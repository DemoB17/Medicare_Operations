# Project Overview

## Introduction
This project analyzes Medicare operations data to evaluate patient demographics, financial performance, physician productivity, and appointment trends.

## Objectives
- Improve operational visibility
- Monitor healthcare KPIs
- Identify patient trends
- Evaluate doctor performance
- Analyze revenue and payment status

## Tools Used
- Microsoft Excel
- SQL
- Power BI
- DAX

## Outcome
An interactive dashboard providing healthcare administrators with actionable insights for operational decision-making.
