# Data Cleaning Process

The dataset was cleaned in Microsoft Excel before analysis.

## Cleaning Steps

- Removed duplicate records.
- Checked for missing values.
- Corrected inconsistent spellings.
- Standardized date formats.
- Converted numerical fields to appropriate data types.
- Verified payment values.
- Validated patient ages.
- Removed unnecessary columns.
- Checked for blank records.

The cleaned dataset was exported as CSV and imported into SQL and Power BI.
